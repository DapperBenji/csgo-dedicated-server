// MANUAL PARA ADMINISTRADORES DE SERVIDORES DE CS:GO DO TLK SERVER

USE O COMANDO sm_rcon ANTES DE TODOS OS OUTROS COMANDOS. ELE QUE DA PERMISSAO DE ADMINISTRADOR PARA RODAR COMANDOS NO CONSOLE.

Digite no console:

MOVENDO JOGADOR DE LUGAR PARA ORGANIZAR A PARTIDA:
* sm_spec <nickdojogador> - Mudar jogador para espectadores imediatamente
* sm_team <nickdojogador> <numero do time>
* sm_swapteam <name> - Mudar jogador a equipe oposta imediatamente
* sm_swapteam_death <name> - Alterne jogador a equipe oposta quando morrem
* sm_swapteam_d <name> - Alterne jogador para a equipe oposta quando a rodada termina

COMANDOS DO MIX (Warmod):
COMANDOS DE ADMINS DO MIX:
* /notlive // Cancels half and sets all as unready. Atalhos: /nl /cancelhalf /ch
* /cancelmatch // Cancels half and sets all as unready. Atalhos: /cm
* /forceallready // Forces all players to become ready. Atalhos: /far
* /forceallunready // Forces all players to become unready. Atalhos: /faur
* /forcestart // Starts the match regardless of player and ready count. Atalhos: /fs
* /forceend // Ends the match regardless of status. Atalhos: /fe
* /minready <num> // Set or display the wm_min_ready console variable
* /maxrounds <num> // Set or display the wm_max_rounds console variable
* /swap // Manually swaps all players to the opposite team
* /t & /ct // TROQUE O NOME DAS EQUIPES
* /knife // ATIVE A RODADA DE FACA. Atalhos: /ko3
* /cancelknife // CANCELE A RODADE DE FACA. Atalhos: /ck
* /readyup // If wm_autoready is 0, this command will be required to start/stop the ready system. Atalhos: /ru
* /readyon // Turns on or restarts the ReadyUp System. Atalhos: /ron
* /readyoff // Turns off the ReadyUp System if enabled. Atalhos: /roff

COMANDOS DOS JOGADORES DO MIX:
* /ready // Marks you ready if /readyup has been done or wm_autoready is 1. Atalhos: /rdy /r
* /unready // Marks you not ready if /readyup has been done or wm_autoready is 1. Atalhos: /notready /unrdy /notrdy /ur /nr
* /scores // Displays score if live. Atalhos: /score /s
* /info // Display who the ready panel if closed. Atalhos: /i
